# react-native-usebottomsheet

Interactive, hooks first React Native Bottom Sheet

## Installation

```sh
npm install react-native-usebottomsheet
```

## Usage

```js
import { multiply } from 'react-native-usebottomsheet';

// ...

const result = await multiply(3, 7);
```
